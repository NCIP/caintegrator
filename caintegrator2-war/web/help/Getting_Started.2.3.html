Using Online Help
The online help explains how to use all of the features of caIntegrator2.
To access online help, use any of the following methods:
•
Click the help menu option on the top menu bar to open the complete online help project.
Note:
You can access online help without being logged in.
•
Click the help icon at the top of each page to open a context-sensitive topic. Context-sensitive help displays information that corresponds to the page from which help was opened.
Once you are in online help, several options help you locate topics of interest.
