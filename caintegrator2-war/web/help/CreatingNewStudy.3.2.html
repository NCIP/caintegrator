Creating a Study – Overview
You can create a caIntegrator2 study by importing clinical study data, genomics data and imaging data. As you create the study, you define its structure in the process, identifying the data sources and mapping the data between different source data. caIntegrator2 has KM-plot as well as genePattern integration for performing analyses.
•
Clinical study data are always in spreadsheet format.
•
Genomic data are expression data from caArray.
•
Imaging data consists of NCIA images or annotations for images, which are in spreadsheet format
