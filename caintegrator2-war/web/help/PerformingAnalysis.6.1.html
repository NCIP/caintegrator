Data Analysis
Once data mapping is complete, user configures the analysis tools from a selection menu. For example – selects the clinical outcome (survival, performance scores etc) for a K-M plot. User can pre-configure Gene-Pattern workflows to be made available to the end user. Link-outs to 3rd party tools either within caBIG or external should be made possible. Simple link-outs with well-structured query strings are preferred.
Data analysis and visualization will be performed using
•
Integration with Gene-Pattern
•
Integration with GeWorkbench (Web version)
•
Existing caIntegrator2 analysis capabilities.
Much about these kinds of analysis in the I-SPY and Rembrandt documentation.
