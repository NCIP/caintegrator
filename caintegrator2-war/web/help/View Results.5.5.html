Browse Imaging Data Results
The system displays a basic search interface and a result set of images and the image features selected as study definition time for inclusion in the image browsing screen. Initially, the entire list of study images will be displayed. The research investigator may view complete details for an image. Details shown for the image include:
•
All image details (name, size, etc.)
•
The series that the image belongs to
•
All image feature attributes
•
The subject ID. Clicking the subject ID will display the detail information described above in Browse Clinical Data.
