Gene Expression Value Plot for Annotation
To generate a gene expression plot, follow these steps:
1.
Select the study whose data you want to analyze in the upper right portion of the caIntegrator2 page. (You must select a study which has genomic data.)
2.
Under Analysis Tools on the left sidebar, select Gene Expression Plot. This opens a page with three tabs
3.
Select the For Annotation tab.
4.
Enter or select fields as described in the following table..
Field
Description
Gene Symbol
Enter one or more gene symbols in the text box or click the icons to locate genes in the following databases. If entering more than one gene in the text box, separate entries by commas.
CGAP – Use this directory to identify genes. Before clicking this link you must enter gene symbols in the text box. This link does not pull anything into caIntegrator2 but does provide information about the gene(s).
caBio – This link pulls identified genes into caIntegrator2 for analysis. Click the icon, enter Keyword(s) in the text box that opens and click Search.
Use the check boxes to identify the genes whose symbols you want to use in the gene expression analysis.
Click the Use Genes button at the bottom of the page. This pulls the checked genes into the Gene Expression ... tab.
Reporter Type
Select the radio button that describes the reporter type:
Reporter ID – Summarizes expression levels for all reporters you specify.
Gene Name – Summarizes expression levels at the gene level.
Sample Groups
Annotation Type – Select the annotation type. Selections are based on the data in the chosen study
Annotation – Select an annotation. Fields are based on the annotation type you select. For example, if you choose Subject, then you could select Gender or Radiation Type or any field that would distinguish the patients into groups based upon study values.
Values – Using conventional selection techniques, select one or more values which will be the basis for the plot. Permissible (available) values or “No Values” correspond to the selected annotation.
Add additional group...all other subjects
Check the box to create an additional group of all other subjects that are not in selected query groups.
Add additional group...control group
Check the box to display an additional group of control samples for this study.
5.
Click the Create Plot button. caIntegrator2 generates the plot which then displays below the Gene Expression Plot criteria in bar graph format. Legends below the plot indicate the plot input.
6.
By default, caIntegrator2 displays a plot showing the mean of the data. You can recalculate the data display by clicking the Plot Type above the graph. See Understanding a Gene Expression Plot
7.
You can modify the plot parameters and click the Reset button to recalculate the plot.
