Launching a Search
To perform a simple search of all annotations in a study, follow these steps:
1.
In caIntegrator2, in the upper right hand corner, select the study you want to browse or perform a simple search.
2.
On the left sidebar, under the first section that displays the [study name], click Search [Study Name]. This opens a simple search query page with five tabs.
3.
On the Criteria tab, select the type of data you want to search. The listed options reflect the type of data you have uploaded to the study.
•
Clinical
•
Gene Expression
•
Image Series
4.
Click the Add button to define data elements for the search.
Clinical and Image Series
5.
For Clinical or Image Series data, an additional drop-down list displays data elements that are annotation definitions (column headers) you specified when you uploaded the data. Select a search criterion from among the options. You can make only one selection at a time.
6.
Additional clinical search criteriaEach choice opens other fields relevant to the selection where you can further define your search query. For example, if you select the Age data element for Clinical data, you can then select age ranges, which are the permissible values you set when you assigned the annotations ().
In some cases, for example, for Laboratory Batch (clinical) or Tumor Location (image), you can manually enter text that defines the values. For most data types, however, you must select pre-defined values shown in the UI fields.
If only a Clinical/Imaging Mapping file was uploaded when the study was created, you cannot enter image search criteria. The search results will, however, display a link that allows you to view the associated images in NCIA.
Gene Expression
7.
For the Gene Expression selection, the only options are Gene Name and Fold Change. Fold change is used to identify genes with expression differences compared to normal samples, as defined when the study was deployed in caIntegrator2. You can enter query values in greater/lesser-than-or-equal-to arguments.
8.
To add additional criteria for the search, repeat steps 3 through 7, as appropriate. You can set more than one data type or more than one criterion for a data type. The criteria become cumulative, thus refining the search.
9.
Once you have configured the query criteria, select the Boolean Or or And search operator at the bottom of the page.
•
Or finds data with at least one of the search criteria
•
And finds data with both/or all search criteria.
10.
Click the Remove button to clear any data elements you have defined.
11.
You can launch the search from this tab. Click the Run Search button. For information about the search results, see
– or –
12.
You can specify the columns you want to display in the search results data. By default, only the column showing the subject/patient identifier displays. For more information, see Columns and Sorting Tabs. On the Sorting tab, you can specify how the data is to be sorted.
As long as you are still in the current query session, you can return to the Criteria, Columns and Sorting tab to modify data and display criteria. If you save the query, your current criteria are saved.
