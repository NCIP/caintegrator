Creating a PatientDID List from a Clinical or p53 Report
On any Clinical (or the p53) page, you can select and save patients to a PatientDID list.
1.
There are two ways to select patients on the Clinical window:
To select an individual, select the box in the I-SPY ID column.To select all of the patients, select the Check All box . To clear all of the patients, uncheck the Check All box.
2.
To save the patients to a PatientDID list, enter a name for the list (or leave the default name by which you named the query).Click the Save Patients button. Sample List Saved appears.
3.
Click the OK button. The new PatientDID list is now displayed in red in the right side bar at the bottom of the PatientDID names. Mouse over the name and the data items appear.
