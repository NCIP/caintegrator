Adding Clinical Data
The Edit Study page opens after you save a new study or click to edit an existing study.
To add or edit clinical metadata in this page, follow these steps:
1.
To add clinical data, on the Edit Study page click the Browse button. Navigate to locate the file. Files must be in CSV file format.
In the Clinical Data Sources section, if a file has already been selected, its information displays in the varying fields.
2.
Click Add Clinical Data File. This opens the Define Fields for Clinical Data page.
The page displays column headers and data values in the file you have designated.
The File Header column on the Define Fields... page displays current column headers. You must designate one column in the file as a unique “identifier” column type. You must also review and define column annotation definitions for each column header in the file. Assigning the Identifier and the annotations are the MOST important steps in creating a new study in caIntegrator2.
If someone has already used the same column header in other files already in caIntegrator2, a term such as “age” or “survival” appears in the Field Definition column above the blue Change Assignment link. If the area above the blue Change Assignment link is blank, it will be populated once you specify the field type.
3.
You must indicate the unique identifier of choice. On the row showing the column header (PatientID in the figure, but other examples are subject identifier, sample identifier, etc), click Change Assignment in the Field Definition column.
