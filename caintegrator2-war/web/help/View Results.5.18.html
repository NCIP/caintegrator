IHC Level of Expression Search Results
An IHC Level of Expression report displays the patients selected from the IHC Level of Expression search. On the IHC Level of Expression page, you can select and save patients to a PatientDID list to further filter queries. You can also sort the patients using the red triangles in the column name.
