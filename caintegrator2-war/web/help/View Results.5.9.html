Viewing imaging results
Cai2 search results will be modified to include a column containing one checkbox per row. The checkboxes will work in conjunction with a button at the bottom of the results page which says “View images in NCIA”, which will open the NCIA site and the NCIA image basket will be loaded with images. Clicking on “View images in NCIA” will open up a new browser window and will display the NCIA website basket which will contain all of the image series that we have requested. In the event that the cai2 study was NOT configured with image annotation for an image series, cai2 will send NCIA a list of Study Instance UIDs, for which NCIA will add all corresponding image series to basket. In the event that the cai2 study was configured with annotation for an image series, we will send NCIA a list of Image Series IDs, for which NCIA will add all corresponding image series to basket.
Cai2 search results will be modified to include a column containing one checkbox per row. The checkboxes will work in conjunction with a button at the bottom of the results page which says “Download images from NCIA”, which will use NCIA grid transfer.Clicking on “Download images from NCIA” take a list of either subjectIDs or StudyInstanceUIDs or ImageSeriesIDs (based on the mapping used for Creating Study) as input to call NCIA's grid transfer.
A zip file organized by patientID, StudyInstanceUID, ImageSeriesUID and image can be downloaded and saved.
1) Open a study that has imaging data associated with it, and who's pointing to production NCIA server (small-study doesn't currently). 2) Make a query that will have image series or patients who are associated to Image Studies and select X amount of those patients in the checkbox. 3) Click the "Retrieve Dicom Images" button 4) Verify a new page pops up, and that a job is running, and eventually after it completes a download button occurs. 5) Download zip file and verify it contains proper dicom files for the given study/series.
