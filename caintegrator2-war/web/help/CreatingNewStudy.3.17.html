Managing Platforms
caIntegrator2 supports a limited number of array platforms, all of which originate from Affymetrix. While they do not represent all of the platforms supported by caArray, caIntegrator2 must have array definitions loaded for the platforms it supports, and be able to properly load the data from caArray and parse it.
On the Manage Platforms page, you can identify, add, or remove supported platforms.
To manage platforms in caIntegrator2, follow these steps:
1.
Click Manage Platforms on the left sidebar. The Manage Platforms page that opens lists the platforms caIntegrator2 currently supports, those that the system can pull from caArray.
2.
You can create a study without genomic data, but you cannot add genomic data to a caIntegrator2 study without a corresponding supported array platform. Supported array platform are listed on this page. If you try to do so, an error message informs you of that fact.
3.
Click the Add button.
