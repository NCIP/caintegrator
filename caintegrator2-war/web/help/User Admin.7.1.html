Administering User Accounts
This section describes the process for creating and managing user accounts in caIntegrator2. It also discusses the processes for managing ownership and access to studies in caIntegrator2.
The options for performing user management tasks are visible in caIntegrator2 on the left sidebar of the browser only if you have these Admin privileges.
