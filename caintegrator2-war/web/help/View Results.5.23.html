Creating a Gene List (Select Genes toolbar)
On the Class Comparison page, you can select and save the genes to a Gene list. To create a Gene list, follow these steps:
1.
To select all of the genes in result list, click the Check All box.
2.
To select some of the genes, check the box in the Gene Symbol column.
Note:
To clear the selected genes, click the clear genes link.
3.
To save the selected genes, enter a unique name for the file next to Select Genes, or maintain the current name, which varies based on the type of Statistical Method selected for the analysis.
4.
Click the Save Genes button.
The results are saved.
5.
Click the OK button.
The new Gene list appears in red in the side bar at the bottom of the Gene Lists names. Mouse over the name and the data items appear.
