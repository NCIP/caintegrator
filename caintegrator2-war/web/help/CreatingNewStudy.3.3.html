Source Data Preparation
Before you can create a new study, it is important that you prepare the data you will employ for your study. You can submit source data to be used in caIntegrator2 studies using three different scenarios:
•
Scenario 1 – When existing LSD applications or existing NCI hosted grid compatible applications are used as source for data.
Before using this method, you should perform one or more of the following to ensure that correct data is loaded to the applications.
•
Submit array data to caArray
•
Submit DICOM-based images to NCIA
•
Scenario 2 –When files/spreadsheets are used as source(s) for data.
Before using this method, you should perform one or more of the following:
•
Prepare clinical annotations as a tab file with one patient in each row with multiple columns for patient-related annotations (demographics, outcome, therapy)
•
Prepare image annotations as a tab file.
•
Scenario 3 – When a combination of spreadsheet/files and existing caGrid applications are used as source data. Before using this method, you should prepare the source data with one or more steps from scenario 1 and 2.
