Browse Genomic Data Results
The system displays a basic search interface and a result set of samples and array data files. The result set will display the subject id (linked to the subject detail display), the sample identifier, the time point (if any), the available data types (expression, copy number, etc.) and the array platform. Details shown for samples include:
•
The sample identifier
•
The associated subject identifier
•
The available array data and platforms for the sample
•
Any clinical annotation available for the sample.
Search results for a genomic data search display reporter data in signal-based values.
 
Research Investigators may also browse arrays, which are displayed in a manner similar to samples.
