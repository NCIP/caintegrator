1.    Research Investigator indicates that he wants to create a new analysis job.
2.    Research Investigator will specify the type of job (gene pattern, geWorkbench, etc), the sample ids, reporter ids, and the files associated with the study that correspond to the array data and annotation  data.  It could be possible that there is one file for array data and one file for annotation data per study, so the research investigator might not need to define this, but only the study of interest, this is TBD.
3.    Research Investigator will save the analysis job by giving it a unique name and description.
4.    Research Investigator, at any time, can update an analysis job that is owned by them.
5.    Research Investigator will run the analysis job and retrieve a task ID that links to the Gene Pattern server after the job is completed for use.
