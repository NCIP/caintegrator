GenePattern Analysis
To export data to GenePattern you must select a query that returns a gene expression data matrix.  If there are no queries in this list then you (or the Study Manager) much create at least one such query.
Analyze Gene Expression Data
When you click Export caIntegrator2 will export your data to GenePattern. The first part of that export will be run a job to convert your data to the correct file format.  You will see that job at the upper right of the GenePattern window that will pop up (see below for example).  You can then analyze your data in GenePattern.  See the GenePattern help page for more information.
Note: Other types of data can be exported to GenePattern, such as copy number data or clinical or imaging data.  Each one would need a separate box on the screen above.
