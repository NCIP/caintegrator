Results Overview
The View Results page shows a collection of reports previously viewed in a particular user session. This allows you to compare reports by opening them in separate windows. You can view results generated with the Search function (see Search Results) and the High Order Analysis function (see High Order Analysis Results).
On the Search Results tab, results are sorted lexigraphically according to the selected column. Any pagination controls that are currently implemented, will page thru the newly sorted results, maintaining the new sort order.
