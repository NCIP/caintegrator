GenePattern Analysis Reports
See High Order Analysis Overview
If you have launched a GenePattern Analysis from the High Order Analysis tab, I-SPY directs you to the View Results tab where you can monitor the status of the task you have initiated. From this tab, you can also launch GenePattern itself to perform tasks within that application. (See Performing a GenePattern Analysis on page 35.).
The GenePattern Job Results (lower panel) in the View Results tab indicates that your query has been sent to GenePattern. It displays the number assigned to the job and its current status. When the status shows Completed, the job number appears in the Gene Pattern Modules section in the upper portion of the page.
In the GenePattern Modules section of the page, you can select your data and an analysis method by which GenePattern will analyze it.
1.
Select a completed job by number. If you have more than one job in the queue, select from the drop down list the data to be analyzed.
2.
Select the analysis (described in the following table) that you want GenePattern to perform on your data.
Analysis Method
Description
HeatMap Viewer
Displays values in a heat map format where the largest values are displayed as the reddest (hot), the smallest values are displayed as the bluest (cool), and intermediate values are a lighter color of either blue or red.
Hierarchical Clustering
Genes or other expression data are clustered according to predetermined algorithms.
K-Nearest Neighbors
Classifies a sample by assigning it the label most frequently represented among the k nearest samples.
Comparative Marker Selection
Finds the genes in a dataset that are most closely correlated with the two phenotypes (ALL and AML).
3.
Click Go to launch the analysis.
The GenePattern Job Results section of the page lists the status of the analysis. Once the job shows completed, click the hypertext link to view the result. Your results display based on your analysis selection, as described in the following bullets.
•
If HeatMap viewer was selected as the analysis method, then the GenePattern HeatMap Viewer Applet opens up and display the values in heat map format
•
If Hierarchical Clustering, K-Nearest Neighbors or Comparative Marker Selection was selected, a GenePattern job is launched from I-SPY. The status of the job is displayed as GenePattern Analysis Results.
The Welcome to GenePattern page appears with general instructions on how to use the application. Refer to Recent Jobs on the right sidebar for the tasks that you can perform with the I-SPY data. If your job is not there, click JobResults tab.
For complete documentation on how to use GenePattern, click Help on the blue bar at the top of the window.
For information about performing another GenePattern query, see Performing a GenePattern Analysis on page 35.
