Exporting Data
Investigator selects list of entries from query results set, saves the list giving it unique name and description.
Can export any data list a downloadable CSV text export.
Can edit, delete entries from the list, or the entire list.
Can create list manually or by uploading a text file containing list of IDs. Can share a list.
List types: sumject, image, sample, array data, gene, genomic region (chromosome, start and end), reporters
Can organize lists into folfders and subfolders
