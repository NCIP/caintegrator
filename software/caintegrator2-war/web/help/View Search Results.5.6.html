Expanding Imaging Data Results
In reviewing imaging search results, it is important to understand the hierarchy of submissions in NBIA. For more information, see Relationship of Patient to Study to Series to Images.
If you run the search before configuring column and sort display parameters, only the Subject Identifiers for the patients/images that meet the criteria and a column containing one check box per row display by default.
If your annotation choice on the Columns page identifies annotations such as tumor size or tumor location, the search results display image series subsets that have those annotations. The check boxes work in conjunction with buttons at the bottom of the results page.
You can add more details for images by configuring image annotations on the Results Type tab. Annotations listed there are the column headers in the image series CSV file(s) that were uploaded to the study. Examples of image details include the following:
•
All image details (name, size, etc.)
•
The series that the image belongs to
•
Image feature attributes
•
The subject ID. Click the subject ID under Clinical Annotations on the Results Type tab to display this.
You can set display parameters for the results on the Columns and Sorting tabs. For more information, see Results Type Tab.
