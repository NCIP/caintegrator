Browsing Search Results
Clinical and Imaging Data
Clinical and/or imaging annotations you specified in the search query displays in the search results in tabular format. If you run the search before configuring column and sort display parameters, only the [subject] ID that meet the criteria and a column allowing you to select each row appear on the table.
You can add details for a single subject by configuring them on the Columns tab. Annotations listed there are the column headers in the CSV file(s) that were uploaded to the study. For information about using the Columns tab, see Columns and Sorting Tabs.
