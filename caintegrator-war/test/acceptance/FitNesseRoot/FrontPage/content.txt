!c !3 Welcome to the !-WebFixture-! v. 2.0

This package includes binaries and examples for !-WebFixture-!, an extension to .NET and Java FIT/!-FitNesse-! that implements a customer-friendly language for web testing, utilising Selenium Remote Control. For more information and the full syntax of these fixtures, see http://fitnesse.info/webtest and http://gojko.net/. 

See JavaExamples and DotNetExamples for examples on how to use the fixtures. See HtmlElements for the HTML page used in examples. If you changed the port for !-FitNesse-! or want to use a different browser to run examples, edit the VariableDefinitions page.
